# Git & GitHub Setup Guide for Beginners (Windows + VS Code)

A clean, beautiful, beginner-friendly webpage that teaches **everything** a new developer needs to start using Git and GitHub on Windows with Visual Studio Code.

Perfect for:
- Students in Saylani Mass IT Institute by TAJ Corporation Sukkur 
- Teachers who want to give their class a single link
- Self-learners who get scared by long command-line tutorials
- Team leads who onboard new junior developers

Live Demo → Just open `index.html` in any browser! No server needed.

## What This Guide Covers (Step-by-Step)

1. Installing Git on Windows + creating a GitHub account  
2. Setting your name & email (required before your first push)  
3. Cloning a repository – two easy ways (VS Code GUI + terminal)  
4. Pushing & pulling changes – with VS Code buttons **and** terminal commands  
5. Working from multiple computers (home PC ↔ laptop ↔ office) without losing work  
6. Fixing the “Please fork this repository” message  
7. Solving login/password problems (Personal Access Token + clearing old credentials)

Everything is written in simple, everyday English — no jargon.
