SMIT Batch 01

Hackathon Preparation Guide

Build a stunning, responsive landing page using only HTML & CSS. Follow these rules to score maximum marks!

1. Project Requirements

  • Create a complete landing page (any theme: business, portfolio, education, etc.)
  • Use proper HTML5 semantic elements
  • All styling in external CSS only
  • Fully responsive (mobile → desktop)
  • Must have: Navbar, Hero, Content Section, Footer
  • Use Google Fonts, Font Awesome, hover effects, transitions

2. What You Must Include

  • Semantic tags: <header>, <nav>, <section>, <footer>
  • Clean heading structure (H1 → H2 → H3)
  • CSS Variables, Flexbox/Grid
  • Media queries for responsiveness
  • Hover effects & transitions
  • Alt text on all images

3. Folder Structure (Must Follow)

            project-folder/
            ├── index.html
            ├── style.css

4. Evaluation Rubrics (100 Marks)

4 External CSS Usage No inline styling, proper file linking & organization (css/style.css) 10
5 Use of Fonts & Icons Google Fonts + Font Awesome (or similar) used tastefully 5

7 Consistent Styling Proper typography, spacing, color contrast throughout the page 10
8 Flexbox / Grid Usage Proper layout implementation using modern CSS (no floats or tables for layout) 10
9 Media Queries Fully responsive design for mobile, tablet, and desktop 5
10 Visual Alignment Proper spacing, alignment & structured layout (no misaligned elements) 5
11 Visual Appeal Clean, modern, aesthetically pleasing design 10
12 Working Navigation Internal links navigate correctly, active/hover states work 5
13 Additional Interactivity CSS animations, hover states, transitions, smooth effects 5
14 Extra Creativity Unique animations, strong branding, custom illustrations, “wow” factor 5
Total Score 100

5. Submission

  • Keep push the code time to time only on your designated branch
  • No late submission shall be accepted
  • All images must be inside the folder
  • No errors when opened

Pro Tips

  • Keep it clean, minimal, and modern
  • Use a consistent color palette
  • Test on your phone!
  • Comment your code
